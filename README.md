# 校友相册小程序

这是一个基于微信小程序开发的校友相册应用。

## 项目结构

```
classmate-album/
├── miniprogram/        # 小程序源码目录
├── typings/           # TypeScript 类型定义
├── project.config.json # 项目配置文件
└── package.json       # 项目依赖配置
```

## 开发环境

- 微信开发者工具
- Node.js
- TypeScript

## 开发说明

- 本项目使用 TypeScript 开发
- 遵循微信小程序开发规范
- 使用 ESLint 进行代码规范检查
